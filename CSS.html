CSS é a linguagem de progamação pela qual vamos dar detalhes em nossa pagina, como: cores em nosso textos, cor da pagina, posição de cada imagem e texto, coloração do fundo e etc...
O CSS - Cascading Style Sheets ou Folhas de Estilos em cascatas. O CSS é uma programação muito usada para informar ao computador onde você deseja que cada imagem ou texto fiquem em sua pagina, como exemplo está pagina que você está vendo. Para que eu disesse ao meu computador onde cada imagem ou texto deveria ficar seria necessario o CSS para dizer onde cada texto ou imagem teria qual tamanho, posição e cores. Como na imagem ao lado.
