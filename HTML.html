HTML - HyperText Markup Language ou Linguagem de Marcação de Hipertexto. A linguagem html serve para que você adicione informaçõe em sua paginá, como por exemplo: textos, imagens, títulos, etc... Está linguagem é importante, pois se trata de um programação em que iremos desenvolver o site, porém não é a única.
Nesta linguagem vamos fazer com que a página tenha o maximo de informações possiveis. Lembrando que não pode utlizar o trabalho de outras pesoas para desenvolver o site, tente desenvolver o site utlizando a criatividade, caso deseje ver alguns de meus trabalhos acesse o GitHub e pesquise por Matheusribeiro2. Na imagem ao lado você pode ver a programção de HMTL. O GitHub não foi criado por mim, mas neste site você consegue ver o trabalho de alguma pessoas, incluisve o meu. Caso deseje programar, este site é excelente para que você salve os seus projetos.
